# Unity-MCP Integration Scope

## In Scope

- Update existing Unity skills (engineer, shader-artist, multiplayer) with Unity-MCP integration
- Update AGENTS.md to include Unity-MCP
- Update MCP generator skill
- Create setup guide documentation
- Add runtime AI section
- Create tools quick reference

## Out of Scope

- Modifying Unity-MCP source code
- Creating new Unity skills
- Unity-MCP server development
- Creating sample Unity projects
- Implementing runtime AI features in actual games

## Constraints

- Must maintain backward compatibility
- All changes are documentation/skill updates only
- No C# code changes to Unity-MCP or Forgewright
